Utmaning
--------------

### Stanna vid vägg
Ändra så att sköldpaddan stannar precis vid väggen då kollision med väggen sker. Start på lösning finns. Se till att det funger åt vänster och höger.

### Gå upp och ner
Sköldpaddan ska kunna flytta sig uppåt och nedåt också. 

Gör så att sköldpaddan inte kan gå in i väggen från något håll utan stannar precis vid väggen.

### Labyrint

Lägg till fler väggar, så att sköldpaddan inte kan lämna skärmen. Lägg också till lite inre väggar.

Det skulle kunna se ut ungefär så här:

    xxxxxxxxxxxx
    x          x
    x xxx xxxx x
    x x      x x
    x    xx    x
    xx    x   xx
    x    xx    x
    x          x
    x x      x x
    x xxx xxxx x
    x          x
    xxxxxxxxxxxx

Kontrollera att sköldpaddan inte kan gå in i någon vägg.

### Klassen Cherry
Lägg ut körsbär som sköldpaddan kan äta.

### Klassen Score
Sköldpaddan får poäng då den äter körsbär. Visa poängen.

### Klassen Background
Ordna en kaklad (eng. tiled) bakgrund.
